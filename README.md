# Примеры проектов для аналитика и data scientist

Данные проекты были выполнены в ходе обучения в Яндекс.Практикуме, профессии "Специалист по Data Science".

| Название проекта | Описание | Используемые библиотеки | 
| :---------------------- | :---------------------- | :---------------------- |
| [Прогнозирование температуры стали во время плавки](yandex-practicum/industry) | Для оптимизации производственных расходов, металлургический комбинат ООО «Так закаляем сталь» решил уменьшить потребление электроэнергии на этапе обработки стали. В проекте строится модель, которая будет предсказывать температуру стали.| *pandas, sklearn, lightgbm, catboost* |
| [Обучение модели классификации комментариев](yandex-practicum/wikishop) | Интернет-магазин запускает новый сервис. Теперь пользователи могут редактировать и дополнять описания товаров, как в вики-сообществах. То есть клиенты предлагают свои правки и комментируют изменения других. Требуется инструмент, который будет искать токсичные комментарии и отправлять их на модерацию. | *pandas, transformers* |
| [Исследование технологического процесса очистки золота](yandex-practicum/gold_recovery) | Строится модель машинного обучения для промышленной компании, разрабатывающая решения для эффективной работы промышленных предприятий. Модель должна предсказать коэффициент восстановления золота из золотосодержащей руды на основе данных с параметрами добычи и очистки. Модель поможет оптимизировать производство, чтобы не запускать предприятие с убыточными характеристиками. | *pandas, sklearn* |
